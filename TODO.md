# Portfolio Action Items

Here are optional tasks you can complete to finalize your portfolio site:

## 1. Profile Photo — `public/profile.jpg`
To update your photo on the site, replace `public/profile.jpg` with your personal photo, keeping the exact filename `profile.jpg`. No code change needed.

## 2. CV / Résumé — `public/cv.pdf`
Add your own CV as `public/cv.pdf`, then in the admin panel (or in `components/Hero.tsx`) set `cv_url` to `/cv.pdf`.

## 3. Optional Dependency Cleanup
`axios` and `next-auth` can optionally be cleaned up if unused in future development.
